# Lelylan API Proxy

Lelylan API proxy.


## Requirements

API Proxy is tested against Node v0.10.36


## Installation

    $ git clone git@github.com:lelylan/api-proxy.git && cd api-proxy
    $ npm install && npm install -g foreman
    $ nf start
